비스틀리
마침내 내 책을 읽을 수 있을 만큼 자란 나의 딸 캐서린에게.
새로운 걸 시도하는 건 어려운 일이다.
나를 도와주고, 또 이게 정신나간 생각이 아니라고
용기를 준 다음의 사람들에게 감사를 표하고 싶다.
조이스 스위니(그리고 그녀를 포함한 금요일 그룹의 여러 회원들),
마제타 기어링, 조지 니콜슨, 피비 예, 캐서린 온더,
사비나 김, 안토니아 마키엣.
종종 그림도 없이 여러 가지 버전의 『미녀와 야수』 이야기를 들어준
내 딸 메레디스, 특히 고맙다.
앤더슨 : 뜻밖의 변신 채팅방 첫 번째 모임에 참석한 걸 환영합니다.
앤더슨 : 누구 있나요? 아니면 여기 있다는 걸 인정할 사람 있나요?
뉴욕야수 님이 입장하셨습니다.
앤더슨 : 반가워요, 뉴욕야수님.
앤더슨 : 여보세요? 들어온 거 봤어요, 뉴욕야수님. 소개 좀 해 주시겠어요?
뉴욕야수 : 1빠로 말하고 싶진 않은데……. 다른 사람 없음?
앤더슨 : 앞서 입장한 사람들은 다들 잠수 중인 것 같아요.
뉴욕야수 : 그럼 그 사람들이 1빠로 하라고 하셈.
앤더슨 : 누구 뉴욕야수님에게 인사하고 싶은 사람 없어요?
침묵소녀 : 반가워요, 뉴욕야수님. 그냥 ‘야수님’이라고 불러도 돼요?
뉴욕야수 : 맘대로. 상관없음.
앤더슨 : 침묵 깨서 고마워요, 침묵소녀님……. 말장난 미안해요. 그런데 님은 어떤 종류의 생물이죠?
침묵소녀 : 인어요. 그냥 작은 인어예요.
앤더슨 : 인어로 변신했다고요?
침묵소녀 : 실은 지금은 인어인데, 변신을 해볼까 생각 중이에요. 이 채팅방에서 결정하는 데 도움을 받을 수 있을 것 같아서요.
앤더슨 : 그게 오늘밤 우리가 하려는 이야기죠. 변신한 경험, 어떻게 지금 모습이 되었나……, 그런 거요.
개구락지 : 님도 변신해씀, 앤디(앤더슨의 애칭: 편집자)?
앤더슨 : 음, 아뇨. 하지만 여러분 모두를 도우려고 이 방을 만든 거예요.
뉴욕야수 : 진짜 소녀임, 침묵소녀? 그러니까 여자, 어, 물고기?
개구락지 : 변시니 뭔지도 모름서 어케 우릴 도음
침묵소녀 : 야수님, 진짜 소녀예요. 그리고 인간 소녀가 될까 생각 중이에요.
앤더슨 : 개구락지님, 저는 여러분 같은 타입의 사건을 공부했어요. 광범위하게. 그림형제, 르프랭스 드 보몽(프랑스 여성작가로 <미녀와 야수> 이야기의 완성본 작가로 알려져 있다: 옮긴이), 악사코프(러시아 소설가: 옮긴이), 퀼러쿠치(영국 평론가 겸 소설가: 옮긴이), 월트 디즈니의 작품 등을 바탕으로 해서 「변신이 진정한 사랑에 미치는 효과」라는 논문도 썼죠…….
뉴욕야수 : 어디 살아, 침묵소녀?
침묵소녀 : 엄청 능력자네요, 앤디. 이런 방을 만들어 줘서 고마워요 :)
앤더슨 : 고마워요, 침묵님.
침묵소녀 : 야수님, 전 덴마크에 살아요. 정확히는 덴마크 근처 대서양에요.
뉴욕야수 : 덴마크?
개구락지 : 오타 미안한데 물갈퀴 달린 바로는 타이핑 힘드러서.
침묵소녀 : 덴마크요. 유럽에 있는.
개구락지 : 발로는
앤더슨 : 이해해요, 개구락지님. 여러 사람들이 모두 모여 이야기하면 좋을 것 같아서 방을 만들었어요.
회색곰남 님이 입장하셨습니다.
회색곰남 : 내가 본 여자 둘에 대해서 이야기하고 싶은데.
뉴욕야수 : 덴마크가 어딘진 앎. 저주받은 다음 계속 공부만 해서 인생 사는 거 같지 않음.
앤더슨 : 좋은 의견이네요, 뉴욕야수님. 변신 이후 생활방식의 변화에 대해서도 이야기해 보죠.
뉴욕야수 : 짱 춥겠네, 침묵소녀!
침묵소녀 : 네, 추워요 ㅎㅎ 하지만 물속은 따뜻해요.
회색곰남 : 내가 본 여자 둘에 대해 이야기하고 싶다니까!
뉴욕야수 : 남친 있음, 침묵소녀?
회색곰남 : 여자 1번은 로즈레드(그림형제의 동화 「Snow-White and Rose- Red」의 주인공. 백설공주 동화와 관련이 없는 전혀 다른 내용이다. 그림형제 판 이야기에서는 추운 겨울날 두 자매의 집에 곰 한 마리가 찾아온다. 소녀들은 곰을 잘 돌봐주고, 봄이 되자 곰은 자신의 보물을 훔쳐간 사악한 난쟁이를 찾기 위해 떠난다. 여름 동안 소녀들은 사악한 난쟁이와 몇 번 마주치는데, 수염이 나무 등걸에 끼었거나 그 외 다른 곤란한 상황에 빠져 있는 난쟁이를 구해 주지만 매번 험악한 욕만 들을 뿐이다. 결국 곰은 난쟁이를 쫓아오고, 난쟁이는 소녀들에게 구해 달라고 하지만 곰의 앞발에 맞아죽는다. 곰은 마법이 풀려 왕자로 돌아간다. 스노화이트는 곰 왕자와 결혼하고, 로즈레드는 그의 동생과 결혼한다: 옮긴이). 진짜 죽임!!!
침묵소녀 : 아직 없긴 해요, 야수님. 근데 왜 묻는지 알 것 같은데…….
개구락지 : 내가 젤 어련 건 파리 먹는 거.
회색곰남 : 2번은 스노화이트임.
침묵소녀 : 지금은 솔로인데, 관심 가는 사람은 있어요……. 선원이에요.
회색곰남 : ‘그’ 스노화이트는 아니고. 로즈레드 동생인 다른 사람인데, 조용한 편. 걔도 괜찮음.
개구락지 : 파리시름.
뉴욕야수 : 문제는 나도 여자를 만나야 된다는 것임, 침묵소녀. 날 사랑해 줄 수 있는 여자로.
침묵소녀 : 고맙긴 한데 전 좋아하는 사람이 있어요, 야수님. 요트에 타고 있던 남자앤데, 물에 빠진 걸 제가 구해줬어요.
앤더슨 : 다들 한꺼번에 말하지 않으면 안 될까요?
뉴욕야수 : 하지만 평소엔 말할 사람이 없어서.
개구락지 : 개구리되명 진자 업슴.
앤더슨 : 알겠어요. 어쨌든 주제가 너무 흩어지지 않도록 차례대로 이야기하죠. 이번이 첫 채팅이니까 우리가 어쩌다 이런 상태가 되었는지 이야기하는 게 좋을 것 같아요. 어떻게 변신하게 되었는지.
개구락지 : 수ㅣ움 마녀 열바게 해서.
뉴욕야수 : 미투.
침묵소녀 : 저도 마녀와 계약할까 하는 중이에요. 정확히는 바다마녀요. 목소리를 주고 인간 다리를 얻는 걸로. 그래서 닉에 ‘침묵’을 넣었어요.
뉴욕야수 : 타이핑 진짜 잘함, 침묵소녀.
침묵소녀 : 고마워요, 야수님. 전 발톱이 아니라 손가락이 있거든요.
회색곰남 : 조낸 잘났음.
앤더슨 : 야수님, 님은 어떻게 변신했는지 얘기해 보시겠어요?
뉴욕야수 : 그러기 싫은데.
앤더슨 : 지금 친구들과 함께 있잖아요, 야수님.
회색곰남 : 빨랑 말해야 내가 두 여자애들 이야기를 하지.
뉴욕야수 : 여자를 둘이나 안다고, 왕자님??? 도대체 어디 사는데???
앤더슨 : 이건 소개팅 방이 아니에요, 야수님.
뉴욕야수 : 하나 나한테 넘기지. 츄바카(영화 <스타워즈>에 나오는 거구의 털북숭이 캐릭터: 편집자) 얼굴을 하고 있으면 여자 만나기가 진짜 힘듦! 내 저주를 풀려면 누구 하나 만나야 되는데.
앤더슨 : 도우미 모임이 필요하겠군요. 그래서 이 방을 만든 거예요.
침묵소녀 : 저기, 다 말해 봐요 야수님. 친구들끼리잖아요.
뉴욕야수 : 오케오케. 우선 알아 둬야 되는 건, 내가 야수라는 거임.
개구락지 : 닉에 있자나.
앤더슨 : 끼어들기 없어요 개구락지님.
뉴욕야수 : 그렇긴 하지만, 옛날엔 뚱보 여자애를 보고 ‘짐승’이라고 불렀던 적도 있었으니까. 난 그런 짐승이 아니라, 진짜로 동물임. 털에 발톱에 다 갖고 있음. 겉보기엔 그냥 완전 동물인데, 속은 아님. 속은 아직도 사람임.
회색곰남 : 나도!
뉴욕야수 : 진짜 제일 힘든 게, 야수가 되기 전에 나는…… 진짜 멋있었거든. 쿨하고 인기도 많고 부자고. 학교에서 친구들이 날 왕자로 뽑았을 정도로.
회색곰남 : 뽑아? 왕자를?
개구락지 : 야수가 무슨 왕자가 되지……. 나 저네 왕자엿는데.
뉴욕야수 : 얘기가 좀 긴데.
개구락지 : 나 저네 왕자여슴.
앤더슨 : 시간은 넉넉해요, 야수님. 다 이야기해 보세요.
뉴욕야수 : :-〈 오케. 이게 다 마녀 때문에 일어난 일임.
개구락지 : 다 그러게 시작되는 거임.
